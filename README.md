# qrcode_generator
Générateur de QR Code

- Ne pas oublier d'installer sur le terminal "qrcode" avec : pip install "qrcode[pil]"
- Lorsqu'on vous demande le filename, mettre le nom du fichier voulu et l'extension .png ou .jpg

Votre QR Code est prêt !
